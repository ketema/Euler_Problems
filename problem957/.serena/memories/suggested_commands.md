# Suggested Commands

## Running Scripts
```bash
poetry run python <script_name>.py
```

## Git Commands (Darwin/macOS)
```bash
git status
git log --oneline -20
git show <commit>
```

## File Operations (Darwin/macOS)
```bash
ls -la
find . -name "*.py"
grep -r "pattern" .
```

## Dependencies
```bash
poetry install
poetry add <package>
```
